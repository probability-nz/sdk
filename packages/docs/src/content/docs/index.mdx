[Probability](https://probability.nz) is a tool for playing 3D tabletop games.

## How it works

Once loaded, live game data is kept in an [Automerge document](https://automerge.org/docs/reference/concepts/#documents); essentially, a shared JSON database.

The SDK exposes the doc and "presence" data (temporary player info). It can be used to build plugins, or small web apps connected to the game (AI opponents, rules engines, dice rollers, etc).
